# NodeJS Social Media Microservice 

A scalable social media backend built with **Node.js (Express)**,**Mongodb (Mongoose)**,
**Redis caching**,  **RabbitMQ messaging** and  **JWT authentication**, full containerized with  **Docker** and orchestrated via **Docker Compose** for high performance, event-driven architecture.

--- 

## Features 

- RESTful APIs for posts with media, users, and search
- JWT Authentication and Authorization
- MongoDB database with mongoose ODM
- Redis caching for fast data retrieval
- RabbitMQ for inter-service communication
- Fully containerized with Docker and Docker Compose
- Modular microservices architecture design for scalability

---

## Tech Stack 

- **Node.js (Express)**
- **MongoDB (Mongoose)**
- **Redis**
- **RabbitMQ**
- **Docker & Docker Compose**
- **JWT Authentication**
